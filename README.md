# Customer_Behavior_Analysis
Complete Data Analytics Portfolio Project with end-to-end industry standard Data Analysis of Customer Shopping Trends from Retail Data using SQL, Python and Power BI.


This project demonstrates an end-to-end Data Analytics workflow using Python, SQL, Power BI, and AI-powered reporting tools. The goal of the project is to extract insights from raw data by performing data cleaning, exploratory data analysis (EDA), SQL analysis, dashboard creation, and automated reporting.

The project covers:

Data loading and preprocessing with Python
Exploratory Data Analysis (EDA)
Data cleaning and transformation
SQL analysis using PostgreSQL/MySQL Server
Interactive dashboard creation in Power BI
Automated report and presentation generation using Gamma AI

The dataset used in this project contains structured business data for analysis purposes.

Corrected inconsistent data types

4. SQL Analysis

Performed SQL queries for:

Aggregations
Joins
Filtering

KPI calculations
5. Dashboard Creation

Built an interactive Power BI dashboard including:

KPI Cards
